# Cat Breeds

### Cat Breeds is an android app for search and find cat breeds.

#### Developed with Jetpack Compose

#### Used Libraries
```
- Retrofit
- Room
- Paging3
- Dagger-Hilt
- Coil
- Coroutines
- Gson
```
